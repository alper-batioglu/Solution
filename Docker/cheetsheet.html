Remove all unused objects

docker system prune

Images

List images

docker image ls

Remove image

docker image rmi <Image ID>
docker rmi $(docker images |grep '<Image Name>')

build image

docker build -t <Image Name>
docker build -t <Image Name> -f <DockerfilePath> ./

Start image

docker start ≶Image ID>
docker run -p <OutboundPort>:<InboundPort> <Image Name/Image ID>

Image contents

docker run -it <Image ID> sh

Image contents root

docker run -u root -it <Image ID> sh

Containers

List Containers

docker container ls
docker container ls -all

Enter Container

docker exec -it <ContainerID> sh

Pause / UnPause container

docker container unpause <ContainerID>

Logs of container

docker logs <ContainerID>

Stop container

docker container stop <ContainerID>

prune containers

docker container prune
